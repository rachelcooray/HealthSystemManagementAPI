# Health System API

Welcome to the Health System API repository! This project is a RESTful API for managing a health system, including functionalities for managing patients, doctors, and appointments.

## Features

- Manage patient records
- Schedule and manage appointments
- Manage doctor records
- Comprehensive exception handling
- Modular and maintainable code structure

## Technologies Used

- Java
- JAX-RS (Jersey)
- Maven
- SLF4J (Simple Logging Facade for Java)
